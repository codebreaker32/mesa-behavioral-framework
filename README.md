This repository contains a working prototype of `mesa.experimental.behaviorals` — an architectural exploration of behavioral agent support in Mesa. This is an **evaluation instrument** following Epstein's "ground truth" principle: implement first, evaluate second, rather than speculating about benefits.

## How to run

```bash
pip install -r requirements.txt
```

OR 

```bash
pip install -e git+https://github.com/codebreaker32/mesa@behaviorals#egg=mesa
```
